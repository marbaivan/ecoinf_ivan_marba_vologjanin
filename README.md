##README

Repositorio para la **práctica de reproducibilidad y control de versiones**, asigatura ecoinformática 2025/2026 UGR.

## Estructura general

-   `biblio/`\
    Citas bibliograficas utilizadas para la realización del documento markdown.
-   `Data/`\
    Carpeta dónde están localizados los datos.
-   `images`\
    Carpeta con las imágenes empradas.
-   `informes`\
    Carpeta con los informes exportados a partir del .Rmd, en formato .pdf y en html.
